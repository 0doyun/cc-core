## 챕터

- **[하네스 엔지니어링](/docs/00-start/harness-engineering)** — 용어의 출처, `runtime harness` 와 `workflow harness` 의 구분, 그리고 OMC 가 왜 workflow harness 레이어로 작동하는지까지 한 챕터에 담았습니다.

이 한 챕터만 읽고 나면 다른 카테고리(Slash Commands, Workflows, Ecosystem 등) 가 훨씬 잘 읽힙니다.
