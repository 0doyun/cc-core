## 이 사이트가 다르게 하는 것

|        | 기존 자원                  | **CC CORE**                           |
| ------ | -------------------------- | ------------------------------------- |
| 언어   | 대부분 영어                | **한국어 우선**                       |
| 구조   | 링크 덤프 or 단편 튜토리얼 | **하네스 엔지니어링 프레임으로 통합** |
| 범위   | CC 혹은 OMC 단일           | **CC + OMC + 에코시스템 전체**        |
| 최신성 | 블로그는 시점 고정         | **"What's New" 주기적 갱신**          |
| 깊이   | 공식 문서 수준             | **공식 + 숨은 커뮤니티 팁**           |

## 누구를 위한 사이트?

- Claude Code를 **이미 쓰고 있지만** 채팅창처럼만 쓰고 있는 사람
- OMC, Hooks, Skills, DESIGN.md 같은 이름은 **들어봤지만** 써본 적 없는 사람
- 흩어진 팁을 **한국어로 한 곳에서** 보고 싶은 사람
- _(Claude Code 자체를 처음 설치하는 경우엔 먼저 [공식 문서](https://code.claude.com/docs) 를 권합니다.)_
